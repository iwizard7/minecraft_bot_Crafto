# Примеры команд для тестирования большого дома

## Создание бота
/crafto spawn Alex
/crafto spawn Builder1
/crafto spawn Builder2

## Команды строительства большого дома

### На английском:
@Alex build big house
@Builder1 build big house
@Builder2 build big house

### На русском:
@Alex построй большой дом
@Builder1 построй большой дом
@Builder2 построй большой дом

## Управление строительством

### Остановка:
/crafto stop Alex
@Alex stop

### Проверка статуса:
@Alex status
@Alex what are you doing
@Alex статус
@Alex что ты делаешь

### Проверка инвентаря:
@Alex show inventory
@Alex inventory stats
@Alex покажи инвентарь

## Коллаборативное строительство

### Создать несколько ботов:
/crafto spawn Worker1
/crafto spawn Worker2
/crafto spawn Worker3

### Все строят один дом:
@Worker1 build big house
@Worker2 build big house
@Worker3 build big house

### Результат: все три бота работают над одним домом одновременно!

## Тестовый сценарий

1. Создайте бота:
   /crafto spawn TestBot

2. Дайте команду строительства:
   @TestBot build big house

3. Наблюдайте за процессом:
   - Бот должен начать летать
   - Блоки должны появляться быстро
   - В логах должны быть сообщения о прогрессе

4. Проверьте результат:
   - Дом должен быть 15x8x12 блоков
   - Должны быть окна, двери, мебель
   - Два этажа с лестницей

5. Остановите бота:
   /crafto stop TestBot

## Производительность

### Один бот:
- Скорость: ~20 блоков/секунду
- Время строительства большого дома: ~2-3 минуты

### Три бота:
- Скорость: ~60 блоков/секунду
- Время строительства: ~1 минута

## Известные проблемы

1. Если дом не строится:
   - Проверьте что NBT файл существует
   - Проверьте логи на ошибки
   - Убедитесь что у бота есть место для строительства

2. Если бот застрял:
   - Используйте /crafto stop <name>
   - Создайте нового бота

3. Если блоки не появляются:
   - Проверьте инвентарь бота
   - Система должна автоматически пополнять материалы
